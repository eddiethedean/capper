# Maintainers

This file lists maintainers and describes expectations for maintaining Capper.

## Current maintainer(s)

- Odos Matthews (`@eddiethedean`)

## Expectations

- Review issues and pull requests on a best-effort basis.
- Ensure changes respect the documented public API and compatibility policy in `docs/compatibility.md`.
- Coordinate and perform releases following `docs/release_checklist.md`.
- Prefer deprecation + minor/major releases over silent breaking changes.

For larger breaking changes or new maintainers, updates to this file should be included in the same pull request.
